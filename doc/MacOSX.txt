
                  Compiling with Equalizer on Mac OS X


Install (or upgrade) all dependency packages (ffmpeg glew qt4-mac
openal) from MacPorts using the universal variant:

# sudo port install (or upgrade --enforce-variants) <package> +universal


Use the following configure options to select the proper frameworks and
architecture:

# ./configure --with-equalizer libEqualizer_CFLAGS="-IPATH_TO_EQUALIZER/src/debug/install/include" libEqualizer_LIBS="-LPATH_TO_EQUALIZER/src/debug/install/lib -lEqualizer" libopenal_LIBS="-L/opt/local/lib -framework OpenAL" libopenal_CFLAGS="-framework OpenAL" CFLAGS="-arch i386" CXXFLAGS="-arch i386"
